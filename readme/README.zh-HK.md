## 📖 項目簡介

**GeoChemistry Nexus** 是一款專為地球化學和岩石學研究人員打造的高效工具，旨在解決科研過程中繪圖繁瑣、計算複雜、格式不統一等痛點。

本軟件支援 **Windows 7** 及更高版本作業系統（Win 8/10/11），通過現代化的互動設計，幫助您快速完成從數據導入到專業圖解生成的全過程。

> 🚀 **我們的願景**：構建一個融合基礎地球化學與岩石學圖解、計算功能的一體化平台，顯著提升科研效率，降低技術門檻，讓科學家專注於科學發現本身。

## ✨ 核心特性

* **🎨 強大的自訂繪圖**
    * 內置豐富的繪圖元工具（線條、多邊形、文字、箭頭、函數曲線等）。
    * 支援自訂腳本，靈活設定數據導入與運算規則，滿足深度客製化需求。

* **🌐 原生多語言支援**
    * **圖解國際化**：同一範本支援中英等多語言一鍵切換，一份成果，全球發布。
    * **介面本地化**：軟件介面全面支援英語（US）、簡體中文、德語等。

* **☁️ 雲端範本生態**
    * 內置由官方與社群共同維護的圖解範本庫。
    * 支援動態更新，無需升級軟件即可實時獲取最新的科研圖解範本。

* **🤝 便捷的分發與協作**
    * 支援將自訂圖解範本打包導出，輕鬆實現跨團隊、跨機構的科研成果共享。

* **🧮 專業地質計算模組**
    * 整合單礦物等多種類型的地質溫度計 (Geothermometer) 範本，滿足多樣化的參數計算需求。

## ⚡ 快速開始

### 下載與安裝

請前往 GitHub **[Releases](https://github.com/MaxwellLei/GeoChemistry-Nexus/releases)** 頁面下載最新版本的安裝包。

### 系統要求
* **作業系統**: Windows 7 SP1 或更高版本 (推薦 Windows 10/11)
* **運行環境**: .NET 6.0 Runtime (如未安裝，軟件通常會提示或自動引導)

如需詳細的使用教學，請瀏覽我們的 [官方文檔站點](https://geochemistry-nexus.pages.dev/)。

## 🗺️ 路線圖 (Roadmap)

我們要做的僅僅不僅是一個工具，而是一個科研生態。以下是我們正在推進的計劃：

- [ ] **構建科研社群**: 搭建專屬社群，支援用戶上傳/分享圖解，提供論壇與反饋板塊。
- [ ] **擴展計算工具箱**: 持續整合更多地球化學常用演算法與模型。
- [ ] **機器學習整合 (ML)**: 引入常用 ML 演算法，輔助進行多維數據分析。
- [ ] **「新圖解」模型支援**: 支援加載基於 ML 的判別模型，讓複雜的 AI 判別像使用三角圖一樣簡單。
- [ ] **AI 智能科研助手**:
    - [ ] Phase 1: 基於 RAG 的智能問答與解決方案生成。
    - [ ] Phase 2: 自動化的數據處理流水線與分析代理。
