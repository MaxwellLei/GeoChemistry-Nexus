# Settings

The software's settings section is very diverse and will continuously be updated with more personalization options as features expand.

## General Program Settings

Settings are mainly divided into four parts: file paths, internationalization & personalization, message notifications, and others. Below is a brief description of the main settings:

![tutorial_settings1](/img/v0.7.1/tutorial_settings1.webp)

### 🌍 Language & Appearance

-   **Language**: Currently supports Simplified Chinese, Traditional Chinese, English, German, Spanish, Japanese, Korean, and Russian.
-   **Startup background image**: The background image displayed when the software starts.

:::info

Regarding language support, if you would like to help with localization for your country, we warmly invite you to join our translation work! Most current translations are AI-assisted, which means some parts may be inaccurate—your help is very much needed and appreciated.

:::

### 🗨️ Message Notifications

-   **Auto-close duration**: The duration after which message notification popups automatically close while the software is running. Please note that error and warning messages require manual closing.

### ⚙️ Other

-   **Start with system**: The software will automatically start when the system starts.
-   **Developer mode**: This mode provides debugging features for developers. Non-developers are advised not to enable it to avoid accidental operations.
-   **Auto-check for updates**: The software will automatically check for updates on startup.
-   **Exit method**: The software can be exited in two ways: exit directly or minimize to the system tray. If you choose minimize to tray, you can right-click the tray icon and select exit to close the software.

## Plotting Module Settings

Plotting module settings are specifically for features within the plotting module, including but not limited to:

*   Third-party program path configuration
*   Default third-party program
*   Default chart loading language
*   Automatic updates for chart templates and category structure
