# 🔢 自定義函數

對於地質溫度計的計算，我們也可以通過內置的自定義函數來直接計算。如下是我們目前內置的地質溫度計計算函數：

| 函數                                                         | 參數數量 | 說明                                               |
| ------------------------------------------------------------ | -------- | -------------------------------------------------- |
| `Zircon_Loucks_et_al_2020(ti,p,aTiO2,aSiO2)`                 | 4        | 鋯石 Ti 溫度計算，主量, Loucks et al. (2020)       |
| `Zircon_Watson_and_Harrison_1983(SiO2,Al2O3,fe2o3,feo,mgo,p2o5,cao,k2o,na2o)` | 10       | 鋯石 Zr 溫度計算，主量, Watson and Harrison (1983) |
| 待編寫...                                                    |          |                                                    |

> 需要注意的是：由於地質溫度計中間計算過程的考慮以及溫度計計算方法的擴展，函數將會隨著版本的更新有一定的變動。使用前建議優先參考最新文檔。
