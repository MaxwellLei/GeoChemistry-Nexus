# 🏠 主頁

# 目標

主頁的初始設計目標是整合地球科學領域常用網站的快捷入口，並提供實用的小組件，以支持相關操作與計算，最終促進科研效率的提升。

![plot_home_ui](imgs/plot_home_ui.png)

主頁主要是承載：**快捷鏈接**和**小組件**。

當然，目前情況下，默認軟件顯示當前系統日期，時間。提供一個搜索欄，該搜索欄目前只是一個簡單的鏈接跳轉。在該搜索欄搜索的內容會跳轉本地瀏覽器的谷歌搜索進行搜索。

> 搜索功能暫不可用

:::info

隨著後續軟件的更新，主頁的功能將會更加實用。

我們歡迎大家在使用過程中提供寶貴的反饋，以提高軟件的可用性和便利性。🌹

:::
