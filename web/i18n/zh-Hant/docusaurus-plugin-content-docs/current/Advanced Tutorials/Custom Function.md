# 🔢 自定義函數

對於地質溫度計計算，你還可以利用內建的自定義函數進行直接計算。目前內建的地質溫度計計算函數如下：

| 函數 | 參數數量 | 描述 |
| :--- | :--- | :--- |
| `Zircon_Loucks_et_al_2020(ti,p,aTiO2,aSiO2)` | 4 | 鋯石 Ti 溫度計算，主量元素，Loucks et al. (2020) |
| `Zircon_Watson_and_Harrison_1983(SiO2,Al2O3,fe2o3,feo,mgo,p2o5,cao,k2o,na2o)` | 10 | 鋯石 Zr 溫度計算，主量元素，Watson and Harrison (1983) |
| 待定... | | |

> **注意：** 出於對中間計算過程和溫度計計算方法擴展的考慮，這些函數可能會隨著版本更新而更改。建議在使用前查閱最新文件。
