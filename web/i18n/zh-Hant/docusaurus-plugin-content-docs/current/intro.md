# 🌍 簡介

**GeoChemistry Nexus 是面向地球化學與岩石學研究者的一體化工作台**：把資料預處理、判別圖解繪製、地溫計與 CIPW 計算、PNG/SVG 匯出收攏到同一介面；並透過雲端模板生態，讓圖解與演算法持續更新、便於分享。

讓我們在 **5 分鐘** 內開始使用 **GeoChemistry Nexus** ⏱️。

## 前置條件 🛠️

需要提前說明的是：**軟體僅適用於 Windows 使用者**。

目前，尚無針對 Linux 和 Mac 的詳細說明。建議優先使用虛擬機執行程式。我們計畫在未來遷移到 `Avalonia` 以實現完全的跨平台相容性。 📅

:::tip

**Windows 使用者**：請確保你的作業系統版本至少為 Windows 7。

:::

## 以此開始 🚀

[點擊這裡](https://github.com/MaxwellLei/GeoChemistry-Nexus/releases) 造訪下載頁面。選擇合適的版本，下載安裝。然後，按兩下執行 `GeoChemistry Nexus.exe` 🖱️即可。
