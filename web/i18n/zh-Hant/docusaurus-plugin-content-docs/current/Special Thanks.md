# 🙏 特別鳴謝

GeoChemistry Nexus 的存在歸功於社群的支援。

## 貢獻者

感謝每一位透過 GitHub Issue、討論區或電子郵件參與本專案的人，包括但不限於：

* 回報 bug、提出功能建議
* 貢獻程式碼、測試與文件
* 分享或共建判別圖解、溫壓計等模板

* 參與多語言本地化等等

所有經許可公開的貢獻者，將收錄於專案貢獻者名單；若你有個人首頁或相關連結，歡迎告知我們一併展示。
