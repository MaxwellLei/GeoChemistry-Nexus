# 🛣️ 路線圖

GeoChemistry Nexus 的目標不只是一個「畫圖軟體」，而是逐步建成 **資料 · 圖解 · 計算 · 協作** 一體化的地球化學與岩石學科研工具鏈。

## ✅ 已完成


### 核心工作流
- [x] **主頁**：快捷連結、實用小元件
- [x] **資料預處理**：主量氧化物欄識別，Fe 價態估算 / 反算，異常值、缺失值、檢出限處理策略
- [x] **判別圖解**：基於模板庫繪製三角圖、散點圖、蛛網圖等；支援資料投影與樣式編輯
- [x] **地溫計（GTM）**：內建多礦物計算模板及類 Excel 自訂函數
- [x] **CIPW 標準礦物計算**：全岩主量資料 norm 計算與結果匯出
### 平台能力
- [x] **模板生態**：官方模板雲端動態更新；JSON / ZIP 格式自訂模板建立、匯入與打包分享
- [x] **多語言**：介面本地化（中 / 英 / 德等）；圖解模板多語言一鍵切換
- [x] **匯出與整合**：PNG / JPG / BMP / SVG 匯出；支援與 CorelDRAW、Inkscape、Adobe Illustrator 等第三方軟體協作

## 🔭 中長期願景
### 機器學習輔助分析
引入常用機器學習流程，輔助地球化學資料判別與分類，輸出模型評估、變數重要性及 ROC 曲線、混淆矩陣等視覺化結果——**無需使用者自行設定 Python 環境**。
### 「新圖解」式判別模型
支援載入可擴展的機器學習判別模型（如基於大資料訓練的新型判別圖），讓研究者直接應用「新圖解」進行分析，而不必掌握完整的 ML 開發棧。
### AI 科研助手（分階段）
1. **第一階段**：基於內建說明文件與知識庫的 RAG 問答，根據關鍵字與場景給出操作建議  
2. **第二階段**：進一步輔助資料處理與分析流程，簡化重複性操作
