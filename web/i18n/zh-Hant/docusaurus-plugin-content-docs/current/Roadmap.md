# 計劃中

- [ ] **研究社群建設**：我們將建立一個專門的研究社群，支援研究人員上傳和分享自製的科學圖表。社群還將配備問題反饋區和論壇討論模組，以促進成員互動、經驗分享和解決方案改進。
- [ ] **擴展地球化學計算工具**：根據使用者需求，我們將不斷更新和完善與地球化學相關的科學計算方法，提供更豐富的計算功能，以支援多樣化的科研應用場景。
- [ ] **機器學習資料分析**：未來的軟體版本將引入常用的機器學習演算法來輔助地球化學資料分析，為研究人員提供高效智慧的資料處理能力。
- [ ] **「新圖表」風格機器學習模型支援**：隨著大數據和機器學習的快速發展，許多模型已成為新的「判別圖」工具。軟體將內建可擴展介面，支援快速載入和訓練此類模型。研究人員可以直接應用「新圖表」進行分析和判別，無需掌握機器學習或 Python 等技術，避免了複雜的環境配置。
- [ ] **AI 智慧助手**：利用內建的幫助文件和知識嵌入大模型技術，AI 助手將在第一階段能夠根據研究人員提供的關鍵詞和需求生成針對性的解決方案；在第二階段，它將進一步實現自動化資料處理和分析，簡化科研工作流程並提高效率。
