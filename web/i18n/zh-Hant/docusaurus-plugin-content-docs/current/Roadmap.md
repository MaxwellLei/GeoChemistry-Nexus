# 🛣️ 路線

## 🌍 **地球科學圖解 (進行中)** ✏️

集成全球地球科學家開發的多種地質投圖與繪圖方法，形成統一的工具包，支持用戶根據具體需求選擇適用的方法，並實現一鍵式繪圖操作。該模塊不僅加速科研工作中的常規圖件生成過程，還促進各類地質方法和工具的標準化與擴大化應用，提升地質科研的整體效率。

## 🌡️ **溫度計、溫壓計計算 (進行中)** 📊  

集成多種地質溫壓計計算模型，並計算形成溫度、壓力等物理參數。系統提供詳細的計算過程及結果解釋，並輸出相應的相圖和溫壓圖，輔助用戶進行深層次的地質解釋與分析，滿足地質科研與應用中的精細化需求。

## 🤖 **人工智能助手** 💡  

負責對向量化底圖指南中的特定內容進行解析和建議，旨在幫助用戶在面對判別分析不明確的場景下，提出基礎的數據需求和核心數據集。智能助手通過智能化分析，依據用戶提供的數據特徵，推薦最優分析方案。最終目標是簡化操作複雜度，顯著提升分析效率和準確性。

## 📈 **機器學習數據分析** 🔍  

基於機器學習算法，對用戶的地球化學數據進行深入分析。用戶可自定義選擇機器學習模型，系統支持自動化的超參數調優、模型訓練與評估。最終輸出包括模型性能評估、變量重要性分析以及相關可視化圖表，如ROC曲線、混淆矩陣、特徵重要性條形圖等，幫助用戶全面理解分析結果。

## ⚗️ **地球化學相關科學計算** 🧪  

提供針對地球化學數據的專業科學計算工具集，涵蓋元素相關性分析、同位素比值計算、稀土元素分佈模式分析等功能。工具集支持數據批量處理、複雜公式計算及結果可視化，幫助研究人員高效完成地球化學數據的處理與解析，確保科研成果的準確性和可重複性。
