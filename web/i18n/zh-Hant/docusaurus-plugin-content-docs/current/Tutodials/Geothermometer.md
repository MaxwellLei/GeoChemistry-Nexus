# 地質溫壓計

在早期的版本中該模組簡稱為 **GTM**。後來改成地質溫度計，到如今的正式定名為：地質溫壓計。

早期版本包含鋯石、閃鋅礦、石英、磷釔礦、綠泥石和黑雲母等礦物的地質溫度計計算。新版本的地質溫壓計還在更新適配中，不久將會同步。

## 🌟 特性

地質溫壓計模組設計之初就**仿照 Excel 的形式**，來降低使用者的上手成本。**主要是分為兩種形式來使用溫壓計**：

1. **內建雲模板**：預設情況下**內建雲模板**使用者不可修改，由官方維護更新。可以在不更新軟體的情況下，連線取得更多最新的地質溫壓計模板。研究人員可以搜尋並選擇適合自己的模板，並按照指南內容進行資料填寫和計算。**這種方式更直接、更快速，無需程式設計知識**。
2. **獨立視窗**：研究人員可以透過在選單列，檔案 -> 開啟獨立表格，來開啟獨立窗體。在獨立窗體中，研究人員被允許自訂程式設計使用不同的地質溫壓計進行組合計算。具體不同溫壓計對應函數，請參見進階教學部分。**這種方式更複雜，相對靈活**。

當然，地質溫壓計模組也允許研究人員建立自己的自訂溫壓計。透過打包成地質溫壓計包可以匯出分發給其他研究人員使用。

## 地質溫壓計使用範例

### 使用內建模板

首先，在地質溫度計模組內，從選單列中選擇相應的計算模板：

![gtm_ui](/img/v0.7.1/geothermobarometer2.webp)

隨後會出現一個二次確認側邊欄彈窗，確定後之前的模板計算會被重設覆蓋（如果在之前有選擇別的溫壓計計算的話）。

![gtm_ui](/img/v0.7.1/geothermobarometer3.webp)

接下來，會出現一個溫度計的範例模板。使用者只需填寫相應的資料，然後使用「點擊並拖曳」填充柄（類似於 Excel）自動完成計算（如下面的動畫範例所示）。

![](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/Example.gif)

當然，新版本溫壓計已支援檢視內部計算詳情。研究人員可以透過滑鼠選中對應資料列，在底部檢視具體計算中間結果詳情。

![gtm_ui](/img/v0.7.1/geothermobarometer4.webp)

### 獨立視窗-自訂函數

如前面所講述，可以透過在選單列，檔案 -> 開啟獨立表格，來開啟獨立窗體。

獨立視窗表格是一個空表格，包含有一些簡單的工具列按鈕。

![gtm_ui](/img/v0.7.1/geothermobarometer5.webp)

在使用獨立視窗表格就和我們日常**使用 Excel 表格一樣，可以輸入任意數值，透過輸入`=函數名稱(傳遞參數)` 來呼叫函數**。

不同的是，我們的地質溫壓計計算被註冊為了對應的計算函數，研究人員可以根據需求更靈活地自訂內容計算。包括但是不僅限於多溫壓計結果對比。在預設的模板情況下，無法實現同時計算多個溫壓計結果（除非你一個模板一個模板計算）。但是在獨立視窗表格下，你可以直接在同一個表格裡面定義並使用多個地質溫壓計函數來取得計算結果。

:::note

**在獨立視窗表格下，無法檢視對應資料列的中間計算過程**。

:::
