# 🌡️ 地質溫度計

軟件中的地質溫度計頁面簡稱為 **GTM**。該頁面目前包括針對鋯石、閃鋅礦、石英、砷釔、綠泥石和黑雲母等礦物的地質溫度計計算。

:::tip

需要注意的是，目前地質溫度計模塊是一個臨時的 UI，隨著溫度計的功能增強，將會有所調整。

:::

## 🌟 功能

現在我們的地質溫度計模塊使用了類 Excel 的形式，大幅度降低了使用門檻。對於地質溫度計的計算可以通過兩種方式：

1. **直接在單元格使用函數**，方法等同於使用 Excel 函數。我們已經預定義了一些溫度計的計算函數，具體函數內容，需要在高級教程，自定義函數查看。
2. **使用內置的地質溫度計模板**，這種方式更加直接快捷，不需要有編程基礎。

![gtm_ui](imgs/gtm_ui.png)

## 使用示例

### 自定義函數

自定義函數的使用和 Excel 一致，都是在單元格內輸入 `=FucName(args)` 這樣的格式來調用。

具體的自定義函數參考高級教程。

### 使用內置模板

首先，在地質溫度計模塊，在菜單欄選擇對應的地質溫度計計算模板：

![gtm_select_m](imgs/gtm_select_m.png)

在隨後的彈窗中，會確認是否新建表格來參與計算。

如果你的當前表格已經有數據不希望覆蓋的話，請選擇確定。反之，如果需要覆蓋當前表格，選擇取消。

![通知](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/GTM_New_Sheet.png)

緊接著會出現一個溫度計的示例模板，用戶只需要填寫對應的數據，然後使用類似於 Excel 的下拉填充單元格的操作，以完成填充自動計算（如下動圖是示例過程）。

![](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/Example.gif)
