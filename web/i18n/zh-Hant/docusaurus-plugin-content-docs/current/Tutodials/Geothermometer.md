# 🌡️ 地質溫度計

軟體中的地質溫度計頁面簡寫為 **GTM**。此頁面目前包括鋯石、閃鋅礦、石英、磷釔礦、綠泥石和黑雲母等礦物的地質溫度計計算。

:::tip

請注意，當前的地質溫度計模組採用臨時 UI。隨著溫度計功能的增強，它將被調整。

:::

## 🌟 特性

我們的地質溫度計模組現在採用類似 Excel 的介面，顯著降低了學習曲線。可以透過兩種方式進行計算：

1.  **直接在儲存格中使用函數**，與使用 Excel 函數相同。我們為各種溫度計預定義了幾個計算函數。有關具體函數詳情，請參閱高級教學中的「自定義函數」部分。
2.  **使用內建地質溫度計模板**，這種方式更直接、更快速，無需程式設計知識。

![gtm_ui](imgs/gtm_ui.png)

## 使用範例

### 自定義函數

自定義函數的使用與 Excel 一致；只需在儲存格中輸入類似 `=FuncName(args)` 的格式即可調用它們。

具體自定義函數參考請參閱高級教學。

### 使用內建模板

首先，在地質溫度計模組內，從選單列中選擇相應的計算模板：

![gtm_select_m](imgs/gtm_select_m.png)

隨後會出現一個彈窗，詢問是否為計算建立一個新工作表。

如果你當前的工作表包含不想覆蓋的資料，請選擇 **確定**。反之，如果你想覆蓋當前工作表，請選擇 **取消**。

![Notification](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/GTM_New_Sheet.png)

接下來，會出現一個溫度計的範例模板。使用者只需填寫相應的資料，然後使用「點擊並拖動」填滿控點（類似於 Excel）自動完成計算（如下面的動畫範例所示）。

![](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/Example.gif)
