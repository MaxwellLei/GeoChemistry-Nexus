# 資料預處理

**資料預處理（Data Preprocessing）** 頁面用於在後續計算（如 CIPW、繪圖與地球化學指標）之前，對全岩主量元素資料進行快速清洗、歸一化與標準化處理。

![plot_home_ui](/img/v0.7.1/data_preprocessing1.webp)

## 模組功能

該模組主要涵蓋常見的地球化學預處理任務：

- 匯入表格資料並識別氧化物欄
- 按無水基準進行主量元素歸一化
- 估算鐵價態分配（FeO / Fe2O3）
- 計算常用地球化學指標（如 Mg#、A/CNK、A/NK）
- 匯出處理結果用於後續工作流程

## 推薦流程

1. **準備資料表**  
   建議包含標準氧化物表頭，例如 `SiO2`、`TiO2`、`Al2O3`、`FeOT`、`FeO`、`Fe2O3`、`MgO`、`CaO`、`Na2O`、`K2O` 等。

2. **載入資料**  
   可直接貼上資料到表格，或先載入範例資料進行流程驗證。

3. **設定預處理選項**  
   按需求啟用或關閉模組：
   - 無水歸一化
   - 鐵價態估算方法
   - 地球化學指標計算

4. **執行處理**  
   執行預處理並檢視輸出工作表內容。

5. **匯出結果**  
   將結果匯出為 `.csv`，用於後續分析或製圖。

## 關鍵參數說明

### 1）無水歸一化（Anhydrous Normalization）

啟用後可扣除揮發分（如 LOI、H2O、CO2），並將主量氧化物重標定到 100%。

當你需要進行跨樣品可比的主量元素分析時，建議啟用該選項。

### 2）鐵價態估算（Iron Valence Estimation）

當資料僅有總鐵（`FeOT`）時，可根據策略估算 FeO 與 Fe2O3 的分配：

- 從總鐵自動估算
- 基於現有 FeO / Fe2O3 反算
- 使用經驗比例 `Fe3+/Fe` 校正

請根據資料品質與研究物件選擇合適方法。

### 3）地球化學指標（Geochemical Indices）

模組支援計算：

- `Mg#`
- `A/CNK`
- `A/NK`

這些指標常用於岩漿演化與鋁飽和特徵解釋。

## 輸入與輸出

### 輸入

- 包含樣品編號與氧化物欄的工作表
- 建議使用 wt.% 數值格式

### 輸出

- 清洗/歸一化後的欄（與啟用模組相關）
- 鐵價態估算欄（啟用時）
- 地球化學指標欄（啟用時）
- 處理摘要資訊（便於追蹤參數與結果）

## 使用建議

- 建議將原始資料保存在獨立工作表，避免覆蓋。
- 處理前先確認欄名規範，減少欄位識別誤差。
- 用於論文或報告時，建議在方法部分明確記錄所選預處理參數。
