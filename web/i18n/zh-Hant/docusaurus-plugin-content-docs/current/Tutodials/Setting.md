# ⚙️ 設定

軟體的設定部分相當多樣化，隨著功能的擴充，將會持續更新更多個人化選項。

## 一般程式設定

設定主要分為四個部分：檔案路徑、國際化與個人化、訊息通知和其他。以下是主要設定的簡要說明：

![tutorial_settings1](../../static/img/v0.6.1/tutorial_settings1.png)

### 🌍 語言與外觀

- **語言**：目前支援簡體中文、英文、德文、西班牙文、日文、韓文和俄文。
- **啟動背景圖片**：軟體啟動時顯示的背景圖片。

:::info

關於語言支援，如果你願意協助在地化你的國家語言，我們熱烈邀請你加入我們的翻譯工作！目前的翻譯大多由 AI 輔助，這意味著某些部分可能不準確——我們非常需要並感謝你的幫助。

:::

### 🗨️ 訊息通知

- **自動關閉時間**：軟體運作期間訊息通知彈出視窗自動關閉的時間。請注意，錯誤和警告訊息需要手動關閉。

### ⚙️ 其他

- **隨系統啟動**：軟體將在系統開機時自動啟動。
- **開發者模式**：此模式提供開發者的除錯功能。建議非開發者不要啟用，以免發生意外操作。
- **自動檢查更新**：軟體將在啟動時自動檢查更新；但由於伺服器問題，此功能暫時停用，將在未來版本中恢復。
- **退出方式**：軟體可以通過兩種方式退出：直接退出或最小化到系統匣。如果你選擇最小化到系統匣，可以右鍵點擊系統匣圖示並選擇退出以關閉軟體。

## 繪圖模組設定

繪圖模組設定專門針對繪圖模組內的功能，包括但不限於：

* 第三方程式路徑配置
* 預設第三方程式
* 圖表預設載入語言
* 圖表模板和分類結構的自動更新
