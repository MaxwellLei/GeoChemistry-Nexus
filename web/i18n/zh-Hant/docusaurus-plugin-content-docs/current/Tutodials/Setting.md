# 設定

軟體的設定部分非常多樣化，並將隨著功能的擴展不斷更新更多的個人化選項。

## 常規程式設定

設定主要分為四個部分：檔案路徑、國際化與個人化、訊息通知和其他。下面是對主要設定的簡要說明：

![tutorial_settings1](/img/v0.7.1/tutorial_settings1.webp)

### 🌍 語言 & 外觀

-   **語言**：目前支援簡體中文、繁體中文，英語、德語、西班牙語、日語、韓語和俄語。
-   **啟動背景圖像**：軟體啟動時顯示的背景圖像。

:::info

關於語言支援，如果你想幫助進行你所在國家的本地化，我們熱烈邀請你加入我們的翻譯工作！目前的翻譯大多由 AI 輔助，這意味著某些部分可能不準確——非常需要並感謝你的幫助。

:::

### 🗨️ 訊息通知

-   **自動關閉時長**：軟體執行期間訊息通知彈窗自動關閉的時長。請注意，錯誤和警告訊息需要手動關閉。

### ⚙️ 其他

-   **隨系統啟動**：軟體將在系統啟動時自動啟動。
-   **開發者模式**：此模式為開發者提供偵錯功能。建議非開發者不要啟用，以免誤操作。
-   **自動檢查更新**：軟體將在啟動時自動檢查更新。
-   **退出方式**：軟體可以透過兩種方式退出：直接退出或最小化到系統匣。如果你選擇最小化到系統匣，你可以右鍵點擊匣圖示並選擇退出來關閉軟體。

## 繪圖模組設定

繪圖模組設定專門針對繪圖模組內的功能，包括但不限於：

*   第三方程式路徑設定
*   預設第三方程式
*   圖表預設載入語言
*   圖表模板和分類結構的自動更新
