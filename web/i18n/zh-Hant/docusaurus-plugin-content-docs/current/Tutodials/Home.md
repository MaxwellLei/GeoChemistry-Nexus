# 主頁

# 目標

首頁是負責整合地球科學領域常用網站的快捷存取入口，並提供實用的小工具以支援相關操作與計算，從而提升科研效率。

我們的設計目標是盡可能多地整合地質、地球化學及地球科學領域的網站與工具，透過首頁快捷存取連結起來，建構一個互聯互通的便捷通道。

![plot_home_ui](/img/v0.7.1/intro_user_interface_example1.webp)

首頁主要功能是：**公告**，**快速連結** 和 **小元件**。

* **公告**：負責接收伺服器通知
* **快速連結**：負責整合地球科學領域入口，分為兩個大類。**推薦連結**和**個人連結**，推薦連結個人無法修改，是透過伺服器進行內建推薦。當然，如果你希望你的網站被新增到首頁推薦中，請透過「關於」頁面聯絡我們。個人連結是使用者個人進行新增刪除管理個人的喜好連結。
* **小元件**：小元件的設計目標是提供簡單、獨立且基礎的計算或操作功能，以補充常用的外圍工具，方便使用者日常使用。以後會做成可擴展的外掛形式，目前無法自由擴展。
