# 🏠 首頁

# 目標

首頁的最初設計目標是整合地球科學領域常用網站的快速存取，並提供實用的小工具以支援相關操作和計算，最終提高研究效率。

![plot_home_ui](/img/v0.6.1/tutorial_home1.png)

首頁主要顯示 **快速連結** 和 **小工具**。你可以右鍵點擊打開選單列查看相關操作。

關於快速連結，我們的設計理念是使用者可以新增地質學和地球化學中常用的專業網站。此外，軟體還包含內建推薦站點。如果你希望你的網站被新增到首頁推薦中，請透過「關於」頁面聯絡我們。

目前，首頁預設顯示系統日期和時間。它還具有一個搜尋欄，可以搜尋和匹配使用者新增的快速連結和小工具，以便快速尋找和導航。

小工具的設計目標是提供簡單、獨立且基礎的計算或操作功能，以補充常用的周邊工具，方便使用者日常使用。
