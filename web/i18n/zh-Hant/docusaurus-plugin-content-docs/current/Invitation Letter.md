## 🌟 計算方法和功能需求

在地球化學領域，計算方法的準確性和效率至關重要。 **如果您對地球化學計算有深入的知識和經驗，或者您的研究涉及特定的計算方法，我們真誠邀請您為我們的項目做出貢獻**。您的專業知識將有助於增強我們軟件的計算能力，使其更符合研究人員的需求！您的參與也將為新用戶提供寶貴的支持，幫助他們更好地利用我們的工具。

## 💻 開發者

我們的軟件採用 `C#` 開發，基於現代 `MVVM` 架構和 `WPF` 技術，提供良好的可維護性和可擴展性。此外，我們計劃在未來引入 `Python` 來實現一些功能，以開發更靈活和多樣化的功能。 **如果您是 `C#` 或 `Python` 的經驗豐富的開發者，我們熱忱歡迎您加入我們的項目團隊。無論您想貢獻代碼、設計用戶界面，還是進行功能測試，您的參與都會使我們的軟件更好**。
