## 🌟 計算方法和功能需求

在地球化學領域，計算方法的準確性和效率至關重要。**如果你在地球化學計算方面有深入的知識和經驗，或者你的研究涉及特定的計算方法，我們誠摯邀請你為我們的專案做出貢獻**。你的專業知識將有助於增強我們軟體的計算能力，確保它更好地滿足研究社群的需求！你的參與也將為新使用者提供寶貴的支援，幫助他們充分利用我們的工具。

## 💻 開發者

我們的軟體使用 `C#` 開發，基於現代 `MVVM` 架構和 `WPF` 技術，確保了出色的可維護性和可擴展性。此外，我們計劃在未來引入 `Python` 來實現某些功能，以實現更靈活多樣的功能。**如果你是一位經驗豐富的 `C#` 或 `Python` 開發者，我們熱烈歡迎你加入我們的專案團隊。無論你是想貢獻程式碼、設計使用者介面還是進行功能測試，你的參與都將使我們的軟體變得更好**。
