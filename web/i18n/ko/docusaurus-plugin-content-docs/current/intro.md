# 🌍 소개

**GeoChemistry Nexus는 지구화학 및 암석학 연구자를 위한 통합 작업대입니다**: 데이터 전처리, 판별 도해 그리기, 지온계 및 CIPW 계산, PNG/SVG 내보내기를 하나의 인터페이스에 모았습니다. 또한 클라우드 템플릿 생태계를 통해 도해와 알고리즘을 지속적으로 업데이트하고 쉽게 공유할 수 있습니다.

**5분** 안에 **GeoChemistry Nexus** 사용을 시작해 보겠습니다 ⏱️.

## 사전 요구 사항 🛠️

미리 알려 드리자면, **이 소프트웨어는 Windows 사용자만 사용할 수 있습니다**.

현재 Linux 및 Mac에 대한 자세한 안내는 제공되지 않습니다. 가상 머신에서 프로그램을 실행하는 것을 권장합니다. 향후 `Avalonia`로 마이그레이션하여 완전한 크로스 플랫폼 호환성을 구현할 계획입니다. 📅

:::tip

**Windows 사용자**: 운영 체제 버전이 Windows 7 이상인지 확인하세요.

:::

## 시작하기 🚀

[여기를 클릭](https://github.com/MaxwellLei/GeoChemistry-Nexus/releases)하여 다운로드 페이지에 접속하세요. 적합한 버전을 선택하여 다운로드하고 설치합니다. 그런 다음 `GeoChemistry Nexus.exe`를 더블 클릭하여 실행하면 됩니다 🖱️.
