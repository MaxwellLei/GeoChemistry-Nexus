# 지질온압계

초기 버전에서 이 모듈은 **GTM**으로 약칭되었습니다. 이후 지질온도계로 변경되었고, 현재 공식 명칭은 지질온압계입니다.

초기 버전에는 지르콘, 섬아연광, 석영, 린황이트광, 클로라이트, 흑운모 등 광물의 지질 온도계 계산이 포함되어 있었습니다. 새 버전의 지질온압계는 현재 업데이트 및 적응 중이며, 곧 동기화될 예정입니다.

## 🌟 특징

지질온압계 모듈은 처음부터 **Excel 형식을 모방**하여 설계되어 사용자의 학습 비용을 낮췄습니다. **주로 두 가지 형태로 온압계를 사용합니다**:

1. **내장 클라우드 템플릿**: 기본적으로 **내장 클라우드 템플릿**은 사용자가 수정할 수 없으며 공식에서 유지보수 및 업데이트합니다. 소프트웨어를 업데이트하지 않고도 네트워크 연결 시 더 많은 최신 지질온압계 템플릿을 가져올 수 있습니다. 연구자는 적합한 템플릿을 검색·선택하고 가이드에 따라 데이터를 입력하여 계산할 수 있습니다. **이 방식은 더 직접적이고 빠르며 프로그래밍 지식이 필요 없습니다**.
2. **독립 창**: 메뉴 모음에서 파일 -> 독립 스프레드시트 열기를 통해 독립 양식을 열 수 있습니다. 독립 양식에서 연구자는 다양한 지질온압계를 조합하여 사용자 정의 프로그래밍 계산을 할 수 있습니다. 각 온압계에 해당하는 함수는 고급 튜토리얼을 참조하세요. **이 방식은 더 복잡하지만 상대적으로 유연합니다**.

물론 지질온압계 모듈에서는 연구자가 자체 사용자 정의 온압계를 만들 수도 있습니다. 지질온압계 패키지로 묶어보내 다른 연구자에게 배포할 수 있습니다.

![gtm_ui](/img/v0.7.1/geothermobarometer1.webp)

## 지질온압계 사용 예시

### 내장 템플릿 사용

먼저 지질온도계 모듈 내에서 메뉴 모음에서 해당 계산 템플릿을 선택합니다:

![gtm_ui](/img/v0.7.1/geothermobarometer2.webp)

이후 이차 확인 사이드바 팝업이 나타납니다. 확인하면 이전 템플릿 계산이 재설정·덮어쓰기됩니다(이전에 다른 온압계 계산을 선택한 경우).

![gtm_ui](/img/v0.7.1/geothermobarometer3.webp)

다음으로 온도계 예시 템플릿이 나타납니다. 해당 데이터를 입력한 후 「클릭하여 드래그」 채우기 핸들(Excel과 유사)을 사용하여 계산을 자동 완료합니다(아래 애니메이션 예시 참조).

![](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/Example.gif)

새 버전 온압계는 내부 계산 세부 정보 보기를 지원합니다. 연구자는 마우스로 해당 데이터 행을 선택하고 하단에서 구체적인 중간 계산 결과 세부 정보를 확인할 수 있습니다.

![gtm_ui](/img/v0.7.1/geothermobarometer4.webp)

### 독립 창 - 사용자 정의 함수

앞서 설명한 대로 메뉴 모음에서 파일 -> 독립 스프레드시트 열기를 통해 독립 양식을 열 수 있습니다.

독립 창 스프레드시트는 빈 표이며 몇 가지 간단한 도구 모음 버튼이 포함되어 있습니다.

![gtm_ui](/img/v0.7.1/geothermobarometer5.webp)

독립 창 스프레드시트 사용은 **일상적인 Excel 스프레드시트 사용과 같으며, 임의의 값을 입력하고 `=함수이름(매개변수)`를 입력하여 함수를 호출**할 수 있습니다.

차이점은 지질온압계 계산이 해당 계산 함수로 등록되어 연구자가 필요에 따라 더 유연하게 사용자 정의 내용을 계산할 수 있다는 것입니다. 다중 온압계 결과 비교 등을 포함하되 이에 국한되지 않습니다. 기본 템플릿에서는 여러 온압계 결과를 동시에 계산할 수 없습니다(템플릿을 하나씩 계산하는 경우 제외). 그러나 독립 창 스프레드시트에서는 같은 표에서 여러 지질온압계 함수를 직접 정의하고 사용하여 계산 결과를 얻을 수 있습니다.

:::note

**독립 창 스프레드시트에서는 해당 데이터 행의 중간 계산 과정을 볼 수 없습니다**.

:::
