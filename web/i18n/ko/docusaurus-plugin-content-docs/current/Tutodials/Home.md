# 홈

# 목표

홈 페이지는 지구과학 분야에서 자주 사용하는 웹사이트에 대한 빠른 접근 진입점을 통합하고, 관련 작업 및 계산을 지원하는 실용적인 소형 도구를 제공하여 연구 효율을 높이는 역할을 합니다.

우리의 설계 목표는 가능한 한 많은 지질, 지구화학 및 지구과학 분야의 웹사이트와 도구를 홈 페이지의 빠른 접근 링크로 연결하여, 상호 연결된 편리한 통로를 구축하는 것입니다.

![plot_home_ui](/img/v0.7.1/intro_user_interface_example1.webp)

홈 페이지의 주요 기능은 **공지**, **빠른 링크** 및 **위젯**입니다.

* **공지**: 서버 알림 수신을 담당합니다
* **빠른 링크**: 지구과학 분야 진입점을 통합하며, 두 가지 대분류로 나뉩니다. **추천 링크**와 **개인 링크**입니다. 추천 링크는 개인이 수정할 수 없으며, 서버를 통해 내장 추천됩니다. 물론 홈 페이지 추천에 웹사이트를 추가하고 싶으시면 「소개」 페이지를 통해 연락해 주세요. 개인 링크는 사용자가 개인적으로 추가, 삭제 및 관리하는 즐겨찾기 링크입니다.
* **위젯**: 위젯의 설계 목표는 간단하고 독립적이며 기본적인 계산 또는 조작 기능을 제공하여, 일반적으로 사용하는 주변 도구를 보완하고 일상 사용을 편리하게 하는 것입니다. 향후 확장 가능한 플러그인 형태로 만들 예정이며, 현재는 자유롭게 확장할 수 없습니다.

:::info

소프트웨어가 계속 업데이트됨에 따라 홈 페이지 기능은 더욱 실용적이 될 것입니다.

소프트웨어의 사용성과 편의성을 높이는 데 도움이 되는 소중한 피드백을 환영합니다. 🌹

:::
