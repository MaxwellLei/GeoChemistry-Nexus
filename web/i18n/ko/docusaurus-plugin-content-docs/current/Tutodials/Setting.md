# 설정

소프트웨어의 설정 부분은 매우 다양하며, 기능이 확장됨에 따라 더 많은 개인화 옵션이 지속적으로 추가됩니다.

## 일반 프로그램 설정

설정은 주로 네 부분으로 나뉩니다: 파일 경로, 국제화 및 개인화, 메시지 알림, 기타. 아래는 주요 설정에 대한 간략한 설명입니다:

![tutorial_settings1](/img/v0.7.1/tutorial_settings1.webp)

### 🌍 언어 및 외관

-   **언어**: 현재 간체 중국어, 번체 중국어, 영어, 독일어, 스페인어, 일본어, 한국어, 러시아어를 지원합니다.
-   **시작 배경 이미지**: 소프트웨어 시작 시 표시되는 배경 이미지.

:::info

언어 지원에 관해, 해당 국가의 현지화를 도와주고 싶으시다면 번역 작업 참여를 진심으로 초대합니다! 현재 번역의 대부분은 AI 지원으로 작성되었으며, 일부 부분이 부정확할 수 있습니다——도움이 매우 필요하며 감사드립니다.

:::

### 🗨️ 메시지 알림

-   **자동 닫기 시간**: 소프트웨어 실행 중 메시지 알림 팝업이 자동으로 닫히는 시간. 오류 및 경고 메시지는 수동으로 닫아야 합니다.

### ⚙️ 기타

-   **시스템과 함께 시작**: 시스템 시작 시 소프트웨어가 자동으로 시작됩니다.
-   **개발자 모드**: 이 모드는 개발자에게 디버깅 기능을 제공합니다. 비개발자는 오작동을 피하기 위해 활성화하지 않는 것이 좋습니다.
-   **자동 업데이트 확인**: 소프트웨어가 시작 시 자동으로 업데이트를 확인합니다.
-   **종료 방식**: 소프트웨어는 두 가지 방식으로 종료할 수 있습니다: 직접 종료하거나 시스템 트레이로 최소화. 트레이로 최소화한 경우 트레이 아이콘을 우클릭하여 종료를 선택하면 소프트웨어를 닫을 수 있습니다.

## 작도 모듈 설정

작도 모듈 설정은 작도 모듈 내 기능에 특화되어 있으며, 다음을 포함하되 이에 국한되지 않습니다:

*   타사 프로그램 경로 구성
*   기본 타사 프로그램
*   도표 기본 로드 언어
*   도표 템플릿 및 분류 구조 자동 업데이트
