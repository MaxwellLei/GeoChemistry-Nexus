# 사용자 정의 함수

:::warning
현재 이 문서는 업데이트 중 / 아직 완전히 업데이트되지 않았습니다. 잠시만 기다려 주세요.
:::

지질온도계 계산에서는 내장 사용자 정의 함수를 사용하여 직접 계산할 수도 있습니다. 현재 내장된 지질온도계 계산 함수는 다음과 같습니다:

| 함수 | 매개변수 수 | 설명 |
| :--- | :--- | :--- |
| `Zircon_Loucks_et_al_2020(ti,p,aTiO2,aSiO2)` | 4 | 지르콘 Ti 온도 계산, 주량 원소, Loucks et al. (2020) |
| `Zircon_Watson_and_Harrison_1983(SiO2,Al2O3,fe2o3,feo,mgo,p2o5,cao,k2o,na2o)` | 10 | 지르콘 Zr 온도 계산, 주량 원소, Watson and Harrison (1983) |
| 미정... | | |

> **참고:** 중간 계산 과정과 온도계 계산 방법 확장을 고려하여, 이 함수들은 버전 업데이트에 따라 변경될 수 있습니다. 사용 전 최신 문서를 참조하는 것을 권장합니다.
