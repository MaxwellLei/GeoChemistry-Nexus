# 🙏 특별 감사

GeoChemistry Nexus는 커뮤니티의 지원 덕분에 존재합니다.

## 기여자

GitHub Issue, 토론 또는 이메일을 통해 본 프로젝트에 참여해 주신 모든 분께 감사드립니다. 다음을 포함하되 이에 국한되지 않습니다:

* 버그 보고, 기능 제안
* 코드, 테스트 및 문서 기여
* 판별도, 온압계 등 템플릿 공유 또는 공동 구축

* 다국어 현지화 참여 등

공개 허가를 받은 모든 기여자는 프로젝트 기여자 명단에 수록됩니다. 개인 홈페이지나 관련 링크가 있으시면 알려주시면 함께 표시하겠습니다.
