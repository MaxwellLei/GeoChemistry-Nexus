# 🛣️ 로드맵

GeoChemistry Nexus의 목표는 단순한 「작도 소프트웨어」가 아니라, **데이터 · 도해 · 계산 · 협업**이 통합된 지구화학 및 암석학 연구 도구 체인을 점진적으로 구축하는 것입니다.

## ✅ 완료됨

### 핵심 워크플로
- [x] **홈**: 빠른 링크, 실용적인 위젯
- [x] **데이터 전처리**: 주량 산화물 열 인식, Fe 가역 추정 / 역산, 이상치·결측값·검출한계 처리 전략
- [x] **판별도**: 템플릿 라이브러리 기반 삼각도, 산점도, 스파이더도 등 작성; 데이터 투영 및 스타일 편집 지원
- [x] **지온계(GTM)**: 다광물 계산 템플릿 및 Excel 스타일 사용자 정의 함수 내장
- [x] **CIPW 표준광물 계산**: 전암 주량 데이터 norm 계산 및 결과보내기
### 플랫폼 기능
- [x] **템플릿 생태계**: 공식 템플릿 클라우드 동적 업데이트; JSON / ZIP 형식 사용자 정의 템플릿 생성, 가져오기 및 패키지 공유
- [x] **다국어**: 인터페이스 현지화(중 / 영 / 독 등); 도해 템플릿 다국어 원클릭 전환
- [x] **보내기 및 통합**: PNG / JPG / BMP / SVG보내기; CorelDRAW, Inkscape, Adobe Illustrator 등 타사 소프트웨어와의 협업 지원

## 🔭 중장기 비전
### 머신러닝 지원 분석
일반적인 머신러닝 워크플로를 도입하여 지구화학 데이터 판별 및 분류를 지원하고, 모델 평가, 변수 중요도, ROC 곡선, 혼동 행렬 등의 시각화 결과를 출력——**사용자가 직접 Python 환경을 구성할 필요 없음**.
### 「신도해」형 판별 모델
확장 가능한 머신러닝 판별 모델(예: 빅데이터로 훈련된 신형 판별도) 로드를 지원하여, 연구자가 완전한 ML 개발 스택을 익히지 않고도 「신도해」를 직접 적용할 수 있도록 합니다.
### AI 연구 어시스턴트(단계별)
1. **1단계**: 내장 도움말 문서 및 지식베이스 기반 RAG Q&A, 키워드와 시나리오에 따른 작업 제안 제공  
2. **2단계**: 데이터 처리 및 분석 워크플로를 추가로 지원하여 반복 작업 간소화
