# 設定

ソフトウェアの設定部分は非常に多様で、機能の拡張に伴い、より多くのパーソナライズオプションが継続的に追加されます。

## 一般プログラム設定

設定は主に 4 つの部分に分かれます：ファイルパス、国際化とパーソナライズ、メッセージ通知、その他。以下は主要な設定の簡単な説明です：

![tutorial_settings1](/img/v0.7.1/tutorial_settings1.webp)

### 🌍 言語と外観

-   **言語**：現在、簡体字中国語、繁体字中国語、英語、ドイツ語、スペイン語、日本語、韓国語、ロシア語をサポートしています。
-   **起動背景画像**：ソフトウェア起動時に表示される背景画像。

:::info

言語サポートについて、お住まいの国のローカライズにご協力いただける場合は、翻訳作業への参加を心より歓迎します！現在の翻訳の多くは AI 支援によるものであり、一部が不正確な場合があります——ご協力を非常に必要とし、感謝しています。

:::

### 🗨️ メッセージ通知

-   **自動クローズ時間**：ソフトウェア実行中にメッセージ通知ポップアップが自動的に閉じるまでの時間。エラーと警告メッセージは手動で閉じる必要があることにご注意ください。

### ⚙️ その他

-   **システムと同時に起動**：システム起動時にソフトウェアが自動的に起動します。
-   **開発者モード**：このモードは開発者にデバッグ機能を提供します。非開発者は誤操作を避けるため、有効にしないことをお勧めします。
-   **自動更新チェック**：ソフトウェアは起動時に自動的に更新をチェックします。
-   **終了方法**：ソフトウェアは 2 つの方法で終了できます：直接終了するか、システムトレイに最小化します。トレイに最小化した場合は、トレイアイコンを右クリックして終了を選択してソフトウェアを閉じることができます。

## 作図モジュール設定

作図モジュール設定は、作図モジュール内の機能に特化しており、以下を含みますがこれらに限定されません：

*   サードパーティプログラムのパス設定
*   デフォルトのサードパーティプログラム
*   図表のデフォルト読み込み言語
*   図表テンプレートとカテゴリ構造の自動更新
