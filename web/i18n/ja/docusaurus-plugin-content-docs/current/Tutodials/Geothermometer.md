# 地質温圧計

初期バージョンでは、このモジュールは **GTM** と略称されていました。その後地質温度計に改名され、現在の正式名称は地質温圧計です。

初期バージョンには、ジルコン、閃亜鉛鉱、石英、燐イットロム、緑泥石、黒雲母などの鉱物の地質温度計計算が含まれていました。新版の地質温圧計は現在も更新・適応中で、まもなく同期されます。

## 🌟 特徴

地質温圧計モジュールは当初から **Excel の形式を模倣** して設計され、ユーザーの学習コストを低減しています。**主に 2 つの形式で温圧計を使用します**：

1. **内蔵クラウドテンプレート**：デフォルトでは **内蔵クラウドテンプレート** はユーザーが変更できず、公式がメンテナンス・更新します。ソフトウェアを更新せずに、ネットワーク接続時により多くの最新地質温圧計テンプレートを取得できます。研究者は適したテンプレートを検索・選択し、ガイドに従ってデータを入力して計算できます。**この方法はより直接的で迅速であり、プログラミング知識は不要です**。
2. **独立ウィンドウ**：メニューバーの ファイル -> 独立スプレッドシートを開く から独立フォームを開けます。独立フォームでは、異なる地質温圧計を組み合わせてカスタムプログラミング計算が可能です。各温圧計の対応関数については、上級チュートリアルを参照してください。**この方法はより複雑ですが、相対的に柔軟です**。

もちろん、地質温圧計モジュールでは研究者が独自のカスタム温圧計を作成することもできます。地質温圧計パッケージにまとめてエクスポートし、他の研究者に配布できます。

## 地質温圧計の使用例

### 内蔵テンプレートの使用

まず、地質温度計モジュール内で、メニューバーから対応する計算テンプレートを選択します：

![gtm_ui](/img/v0.7.1/geothermobarometer2.webp)

その後、二次確認のサイドバーポップアップが表示されます。確定すると、以前のテンプレート計算がリセット・上書きされます（以前に別の温圧計計算を選択していた場合）。

![gtm_ui](/img/v0.7.1/geothermobarometer3.webp)

次に、温度計のサンプルテンプレートが表示されます。対応するデータを入力し、「クリックしてドラッグ」のフィルハンドル（Excel と同様）を使用して計算を自動完了します（下のアニメーション例を参照）。

![](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/Example.gif)

新版温圧計では内部計算の詳細を表示できます。研究者はマウスで対応するデータ行を選択し、下部で具体的な中間計算結果の詳細を確認できます。

![gtm_ui](/img/v0.7.1/geothermobarometer4.webp)

### 独立ウィンドウ - カスタム関数

前述のとおり、メニューバーの ファイル -> 独立スプレッドシートを開く から独立フォームを開けます。

独立ウィンドウのスプレッドシートは空の表で、いくつかのシンプルなツールバーボタンが含まれています。

![gtm_ui](/img/v0.7.1/geothermobarometer5.webp)

独立ウィンドウのスプレッドシートの使用は、**日常の Excel スプレッドシートの使用と同様で、任意の数値を入力し、`=関数名(引数)` を入力して関数を呼び出せます**。

違いは、地質温圧計計算が対応する計算関数として登録されており、研究者は必要に応じてより柔軟にカスタム内容を計算できることです。複数温圧計結果の比較などを含みますがこれらに限定されません。デフォルトのテンプレートでは、複数の温圧計結果を同時に計算することはできません（テンプレートを一つずつ計算する場合を除く）。しかし独立ウィンドウのスプレッドシートでは、同じ表内で複数の地質温圧計関数を直接定義・使用して計算結果を取得できます。

:::note

**独立ウィンドウのスプレッドシートでは、対応するデータ行の中間計算過程を表示できません**。

:::
