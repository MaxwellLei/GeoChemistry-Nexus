# CIPW 標準鉱物計算

**CIPW 標準鉱物計算**モジュールは、全岩主量元素酸化物分析データを理想化された標準鉱物組合せに変換し、火成岩分類、岩石学比較、シリコン飽和度 / アルミナ飽和度判別に広く使用されます。

CIPW Norm（Cross-Iddings-Pirsson-Washington 標準鉱物）は、4 人の岩石学者によって 1902 年に提案されました。GeoChemistry Nexus は Excel 風テーブルインターフェースで完全な計算フローを実装し、一括計算、診断表示、CSV エクスポートをサポートしています。

![cipw_ui](/img/v0.7.1/cipw1.webp)

## モジュール機能

- 主量元素酸化物データ（wt%）を行ごとに受け取り
- 無水基準に酸化物を正規化
- 鉄の配分を処理（FeO / Fe₂O₃ / FeOT）
- 固定優先順序で酸化物を標準鉱物に配分
- シリコン飽和度、アルミナ飽和状態、質量バランス診断情報を出力
- 後続の作図や分析のため結果をエクスポート

## インターフェース概要

CIPW ページは 3 つの領域に分かれます：

1. **上部ツールバー** — パラメータ設定と操作ボタン
2. **データテーブル** — 酸化物入力、診断列、鉱物結果
3. **下部診断パネル** — 現在選択行の詳細計算情報

### ツールバー操作

| ボタン | 説明 |
| --- | --- |
| **説明** | ソフトウェア内蔵のアルゴリズムフロー説明ウィンドウを開く |
| **サンプル** | 3 行のサンプルデータ（花崗岩、玄武岩、安山岩）を入力 |
| **エクスポート** | 現在のテーブルを CSV ファイルとしてエクスポート |
| **クリア** | すべての入力と計算結果をクリア |
| **計算実行** | 有効なデータを含むすべての行で CIPW 計算を実行 |

### Fe³⁺/Fe 比

**FeOT**（全鉄、FeO 当量で表記）のみが提供される場合、ソフトウェアはユーザー設定の **Fe³⁺/Fe** 比に従って FeO と Fe₂O₃ に分割します。

- 有効範囲：`0` – `1`
- デフォルト値：`0.15`（Le Maitre, 2002）

FeO と Fe₂O₃ の実測値が同時に提供される場合、ソフトウェアは実測配分を直接使用し、FeOT を無視します。

## 入力データ要件

各行は 1 つの全岩試料を表し、入力列に酸化物の**重量パーセント（wt%）**を記入します。

### サポートされる入力酸化物

| 酸化物 | 説明 |
| --- | --- |
| `SiO2`, `TiO2`, `Al2O3`, `MgO`, `CaO`, `Na2O`, `K2O`, `P2O5` | コア主量元素 |
| `Fe2O3`, `FeO`, `FeOT` | 下記の鉄処理ルールを参照 |
| `MnO` | モル比で FeO 当量に変換 |
| `ZrO2`, `Cr2O3` | 副鉱物 / 微量元素酸化物 |
| `CO2`, `S`, `F`, `Cl`, `SO3` | 揮発分組成 |

:::tip

**データ前処理**モジュールで無水正規化、鉄価数推定などのステップを先に完了し、処理後のテーブルを CIPW モジュールに貼り付けて使用できます。

:::

### 鉄入力ルール

| 入力状況 | 処理方法 |
| --- | --- |
| `FeO` と `Fe2O3` を同時提供 | 実測鉄配分を使用 |
| `FeO` または `Fe2O3` のいずれかのみ提供 | 欠損項目を 0 とみなし、警告を表示 |
| `FeOT` のみ提供 | Fe³⁺/Fe 比で FeO / Fe₂O₃ に分割 |
| `FeOT` と `FeO` または `Fe2O3` を同時提供 | 入力不一致、FeOT を無視して警告 |
| 鉄データ未提供 | 0 とみなし、警告を表示 |

`MnO` は計算前に常に FeO 当量に変換されます。

## 推奨フロー

1. **データ準備**  
   酸化物ヘッダーが入力列と一致し、数値が非負の wt% であることを確認します。

2. **データ入力**  
   テーブルに貼り付けまたは入力します。1 行 1 試料、空行は自動スキップされます。

3. **Fe³⁺/Fe 設定**（FeOT 使用時）  
   データセットに異なる鉄酸化仮定が必要な場合、ツールバーでこの比を調整します。

4. **「計算実行」をクリック**  
   ソフトウェアがすべての有効行を処理し、結果を右側の列に書き込みます。

5. **診断情報を確認**  
   結果行を選択し、下部パネルでシリコン / アルミナ飽和度、鉄処理モード、主要鉱物組成、警告を確認します。

6. **結果をエクスポート**  
   完全なテーブル（入力 + 診断 + 鉱物）を CSV としてエクスポートし、保管または後続分析に使用します。

## 出力列の説明

計算結果は入力領域の右側に表示され、区切り列 `│` で分離されます。

### 診断列

| 列名 | 説明 |
| --- | --- |
| **シリコン飽和度** | 過飽和 / 飽和 / 不飽和 |
| **アルミナ飽和状態** | 過アルカリ性 / 過アルミナ質 / 正常アルミナ質 |
| **質量バランス誤差** | 鉱物質量合計と 100% の偏差 |

### 標準鉱物列

鉱物略号は CIPW 慣例に従い、日本語インターフェースではヘッダーに日本語鉱物名が表示されます。一般的な鉱物：

| 略号 | 鉱物 |
| --- | --- |
| `Q` | 石英 |
| `Or`, `Ab`, `An` | 正長石、斜長石（ナトリウム長石）、斜長石（カルシウム長石） |
| `Le`, `Ne`, `Kp` | 白榴石、霞石、カルシライト |
| `Di`, `Hd`, `Wo` | 透輝石、ヘデンバーガイト、珪灰石 |
| `En`, `Fs`, `Fo`, `Fa` | エンスタタイト、フェロシリット、フォスタライト、ファイアライト |
| `Mt`, `Hm`, `Ilm` | 磁鉄鉱、赤鉄鉱、チタン鉄鉱 |
| `Cc`, `Ap`, `Z` | 方解石、燐灰石、ジルコン |

各行には表示閾値を超える含量の鉱物のみ表示されます。

## 診断パネル

計算完了後、任意の結果行をクリックして詳細診断を確認できます：

- **シリコン飽和度** — 不飽和時にハイライト表示
- **アルミナ飽和度** — 過アルカリ性時にハイライト表示
- **鉄処理モード** — 鉄データが実測、推定、欠損のいずれかを表示
- **質量バランス誤差** — 偏差が大きい時にハイライト表示
- **主要鉱物組成** — 含量の降順で並べ替え
- **警告情報** — 鉄データ欠損、FeOT 入力不一致など

ステータスバー右側のボタンで診断パネルを展開、折りたたみ、最大化できます。

## 計算アルゴリズム

計算は古典的な CIPW 標準鉱物フローに従います：

### 1. データ前処理と正規化

- 入力主量元素酸化物データの無水正規化（100% に正規化）
- 鉄配分の処理：ユーザー設定の Fe³⁺/Fe 比に従い FeOT を FeO と Fe₂O₃ に分割
- MnO をモル比で FeO に統合
- 酸化物重量パーセントをモル数に変換

### 2. 揮発分鉱物の形成

優先順序で揮発分組成を消費：

- 方解石（Cc）：CO₂ + CaO
- 蛍石（Fl）：F + CaO
- 黄鉄鉱（Py）：S + FeO
- 岩塩（Hl）：Cl + Na₂O
- 芒硝（Th）：SO₃ + Na₂O

### 3. 副鉱物の形成

- ジルコン（Z）：ZrO₂ + SiO₂
- 燐灰石（Ap）：P₂O₅ + CaO
- クロム鉄鉱（Cm）：Cr₂O₃ + FeO
- チタン鉄鉱（Ilm）：TiO₂ + FeO
- チタン石（Tn）：TiO₂ + CaO + SiO₂
- ルチル（Ru）：残余 TiO₂

### 4. アルミナ飽和状態の判定

- **過アルカリ性（Peralkaline）**：Na₂O + K₂O > Al₂O₃
- **正常アルミナ質（Metaluminous）**：Al₂O₃ ≤ CaO + Na₂O + K₂O
- **過アルミナ質（Peraluminous）**：Al₂O₃ > CaO + Na₂O + K₂O

### 5. 長石とアルカリ金属ケイ酸塩の形成

- 正長石（Or）：K₂O + Al₂O₃ + 6SiO₂
- 斜長石ナトリウム長石（Ab）：Na₂O + Al₂O₃ + 6SiO₂
- 斜長石カルシウム長石（An）：CaO + Al₂O₃ + 2SiO₂
- コランダム（Cor）：残余 Al₂O₃（過アルミナ質の場合）
- アクミタイト（Ac）：Na₂O + Fe₂O₃ + 4SiO₂（過アルカリ性の場合）
- 残余アルカリ金属ケイ酸塩（ns, ks）

### 6. 鉄酸化物の形成

- 磁鉄鉱（Mt）：Fe₂O₃ + FeO
- 赤鉄鉱（Hm）：残余 Fe₂O₃

### 7. 鉄鎂質ケイ酸塩鉱物の形成

- 透輝石（Di）：CaO + MgO + 2SiO₂（マグネシウム端員）
- ヘデンバーガイト（Hd）：CaO + FeO + 2SiO₂（鉄端員）
- エンスタタイト（En）：MgO + SiO₂
- フェロシリット（Fs）：FeO + SiO₂
- 珪灰石（Wo）：残余 CaO + SiO₂

### 8. シリコン飽和度補正

- **過飽和**（SiO₂ 残余）：石英（Q）を形成
- **飽和**（SiO₂ ちょうど使い切り）：石英なし、似長石なし
- **不飽和**（SiO₂ 不足）：優先順序で鉱物を変換 —
  - 紫蘇輝石（En + Fs）→ カンラン石（Fo + Fa）
  - 正長石（Or）→ 白榴石（Le）
  - 白榴石（Le）→ カルシライト（Kp）
  - 斜長石ナトリウム長石（Ab）→ 霞石（Ne）

### 9. 結果出力

- 各鉱物のモル数にモル質量を乗じて標準鉱物質量を算出
- 重量パーセント（wt%）に正規化して出力
- シリコン飽和度、アルミナ飽和状態、質量バランス誤差などの診断情報を報告

:::
