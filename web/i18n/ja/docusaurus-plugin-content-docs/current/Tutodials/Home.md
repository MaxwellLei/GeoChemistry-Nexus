# ホーム

# 目標

ホームページは、地球科学分野でよく使われるウェブサイトへのクイックアクセス入口を統合し、関連操作・計算を支援する実用ウィジェットを提供することで、科研効率を向上させる役割を担います。

設計目標は、地質・地球化学・地球科学分野のウェブサイトとツールを可能な限り統合し、ホームページのクイックリンクで相互に接続し、相互連携の便利なチャネルを構築することです。

![plot_home_ui](/img/v0.7.1/intro_user_interface_example1.webp)

ホームページの主な機能は：**お知らせ**、**クイックリンク**、**ウィジェット** です。

* **お知らせ**：サーバーからの通知を受け取る
* **クイックリンク**：地球科学分野の入口を統合。2 つの大カテゴリに分類。**おすすめリンク**と**個人リンク**。おすすめリンクは個人では変更できず、サーバーによる組み込みおすすめです。ホームページのおすすめにサイトを追加してほしい場合は、「について」ページからお問い合わせください。個人リンクはユーザーが個人的に追加・削除・管理するお気に入りリンクです。
* **ウィジェット**：シンプルで独立した基本的な計算・操作機能を提供し、よく使う周辺ツールを補完して日常利用を便利にすることを目指しています。将来は拡張可能なプラグイン形式にする予定ですが、現時点では自由に拡張できません。
