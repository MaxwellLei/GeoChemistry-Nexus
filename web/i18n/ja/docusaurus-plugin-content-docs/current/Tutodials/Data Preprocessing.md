# データ前処理

**データ前処理（Data Preprocessing）** ページは、後続の計算（CIPW、作図、地球化学指標など）の前に、全岩主量元素データを迅速にクリーニング、正規化、標準化するために使用されます。

![plot_home_ui](/img/v0.7.1/data_preprocessing1.webp)

## モジュール機能

このモジュールは主に一般的な地球化学前処理タスクをカバーします：

- 表形式データのインポートと酸化物列の認識
- 無水基準による主量元素の正規化
- 鉄価数配分の推定（FeO / Fe2O3）
- 一般的な地球化学指標の計算（Mg#、A/CNK、A/NK など）
- 後続ワークフロー用の処理結果のエクスポート

## 推奨フロー

1. **データ表の準備**  
   `SiO2`、`TiO2`、`Al2O3`、`FeOT`、`FeO`、`Fe2O3`、`MgO`、`CaO`、`Na2O`、`K2O` などの標準酸化物ヘッダーを含めることを推奨します。

2. **データの読み込み**  
   データを表に直接貼り付けるか、まずサンプルデータを読み込んでフローを検証できます。

3. **前処理オプションの設定**  
   必要に応じてモジュールを有効/無効にします：
   - 無水正規化
   - 鉄価数推定方法
   - 地球化学指標計算

4. **処理の実行**  
   前処理を実行し、出力ワークシートの内容を確認します。

5. **結果のエクスポート**  
   結果を `.csv` としてエクスポートし、後続の分析や作図に使用します。

## 主要パラメータの説明

### 1）無水正規化（Anhydrous Normalization）

有効にすると、揮発分（LOI、H2O、CO2 など）を差し引き、主量酸化物を 100% に再スケールします。

試料間で比較可能な主量元素分析が必要な場合、このオプションを有効にすることを推奨します。

### 2）鉄価数推定（Iron Valence Estimation）

データに全鉄（`FeOT`）のみがある場合、戦略に従って FeO と Fe2O3 の配分を推定できます：

- 全鉄から自動推定
- 既存の FeO / Fe2O3 に基づく逆算
- 経験比 `Fe3+/Fe` による補正

データ品質と研究対象に応じて適切な方法を選択してください。

### 3）地球化学指標（Geochemical Indices）

モジュールは以下の計算をサポートします：

- `Mg#`
- `A/CNK`
- `A/NK`

これらの指標は、マグマ進化とアルミナ飽和特性の解釈に一般的に使用されます。

## 入力と出力

### 入力

- 試料番号と酸化物列を含むワークシート
- wt.% 数値形式の使用を推奨

### 出力

- クリーニング/正規化後の列（有効なモジュールに依存）
- 鉄価数推定列（有効時）
- 地球化学指標列（有効時）
- 処理サマリー情報（パラメータと結果の追跡用）

## 使用上の推奨事項

- 元データは別のワークシートに保存し、上書きを避けることを推奨します。
- 処理前に列名の規則を確認し、フィールド認識エラーを減らしてください。
- 論文や報告書に使用する場合、方法セクションに選択した前処理パラメータを明確に記録することを推奨します。
