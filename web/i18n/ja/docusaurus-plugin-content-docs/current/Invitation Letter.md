## 🌟 計算方法と機能リクエスト

地球化学分野では、計算方法の正確性と効率が極めて重要です。**地球化学計算に深い知識と経験をお持ちの方、または特定の計算方法に関する研究を行っている方は、ぜひ本プロジェクトへの貢献をご検討ください**。あなたの専門知識は、ソフトウェアの計算能力の強化に役立ち、研究コミュニティのニーズにより適したものにします！あなたの参加は、新規ユーザーに貴重なサポートを提供し、ツールを最大限に活用できるよう支援します。

## 💻 開発者

本ソフトウェアは `C#` で開発され、モダンな `MVVM` アーキテクチャと `WPF` 技術に基づいており、優れた保守性と拡張性を確保しています。さらに、将来的には `Python` を導入して一部の機能を実装し、より柔軟で多様な機能を実現する予定です。**経験豊富な `C#` または `Python` 開発者の方は、ぜひプロジェクトチームにご参加ください。コードの貢献、ユーザーインターフェースの設計、機能テストなど、どのような形でもご参加いただけるとソフトウェアがより良くなります**。
