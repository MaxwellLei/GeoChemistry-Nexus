# カスタム関数

:::warning
現在、このドキュメントは更新中 / まだ完全に更新されていません。しばらくお待ちください。
:::

地質温度計計算では、内蔵のカスタム関数を使用して直接計算することもできます。現在内蔵されている地質温度計計算関数は以下のとおりです：

| 関数 | パラメータ数 | 説明 |
| :--- | :--- | :--- |
| `Zircon_Loucks_et_al_2020(ti,p,aTiO2,aSiO2)` | 4 | ジルコン Ti 温度計算、主量元素、Loucks et al. (2020) |
| `Zircon_Watson_and_Harrison_1983(SiO2,Al2O3,fe2o3,feo,mgo,p2o5,cao,k2o,na2o)` | 10 | ジルコン Zr 温度計算、主量元素、Watson and Harrison (1983) |
| 未定... | | |

> **注意：** 中間計算過程と温度計計算方法の拡張を考慮し、これらの関数はバージョン更新に伴い変更される場合があります。使用前に最新のドキュメントを参照することをお勧めします。
