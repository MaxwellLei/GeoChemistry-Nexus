# 🌍 はじめに

**GeoChemistry Nexus は、地球化学・岩石学の研究者向けに設計された統合ワークベンチです**。データ前処理、判別図の作成、地質温度計・CIPW 計算、PNG/SVG エクスポートを同一インターフェースに集約し、クラウドテンプレート・エコシステムを通じて図解とアルゴリズムを継続的に更新・共有できます。

**5 分**で **GeoChemistry Nexus** を始めましょう ⏱️。

## 前提条件 🛠️

事前にお伝えしておきます：**本ソフトウェアは Windows ユーザーのみを対象としています**。

現在、Linux および Mac 向けの詳細な説明はありません。必要に応じて仮想マシンでの実行をお勧めします。将来的には `Avalonia` への移行により、完全なクロスプラットフォーム対応を目指しています。📅

:::tip

**Windows ユーザー**：オペレーティングシステムのバージョンが Windows 7 以上であることをご確認ください。

:::

## さあ始めましょう 🚀

[こちらをクリック](https://github.com/MaxwellLei/GeoChemistry-Nexus/releases)してダウンロードページにアクセスしてください。適切なバージョンを選び、ダウンロードしてインストールします。その後、`GeoChemistry Nexus.exe` をダブルクリックして実行します 🖱️。
