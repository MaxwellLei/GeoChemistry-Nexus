# 🛣️ ロードマップ

GeoChemistry Nexus の目標は単なる「作図ソフト」ではなく、**データ · 図解 · 計算 · 協力** を一体化した地球化学・岩石学の研究ツールチェーンを段階的に構築することです。

## ✅ 完了済み

### コアワークフロー
- [x] **ホーム**：クイックリンク、実用的なウィジェット
- [x] **データ前処理**：主量酸化物列の認識、Fe 価数の推定 / 逆算、外れ値・欠損値・検出限界の処理戦略
- [x] **判別図**：テンプレートライブラリに基づく三角図、散布図、スパイダー図などの作成；データ投影とスタイル編集をサポート
- [x] **地質温度計（GTM）**：複数鉱物の計算テンプレートと Excel 風カスタム関数を内蔵
- [x] **CIPW 標準鉱物計算**：全岩主量データの norm 計算と結果エクスポート
### プラットフォーム機能
- [x] **テンプレート・エコシステム**：公式テンプレートのクラウド動的更新；JSON / ZIP 形式のカスタムテンプレート作成、インポート、パッケージ共有
- [x] **多言語**：インターフェースのローカライズ（中 / 英 / 独など）；図解テンプレートの多言語ワンクリック切替
- [x] **エクスポートと連携**：PNG / JPG / BMP / SVG エクスポート；CorelDRAW、Inkscape、Adobe Illustrator などのサードパーティソフトとの協業をサポート

## 🔭 中長期ビジョン
### 機械学習支援分析
一般的な機械学習ワークフローを導入し、地球化学データの判別と分類を支援。モデル評価、変数重要度、ROC 曲線、混同行列などの可視化を出力——**ユーザーが自ら Python 環境を設定する必要はありません**。
### 「新図解」型判別モデル
拡張可能な機械学習判別モデル（例：ビッグデータで訓練した新型判別図）の読み込みをサポートし、研究者が完全な ML 開発スタックを習得せずに「新図解」を直接適用できるようにします。
### AI 研究アシスタント（段階的）
1. **第 1 段階**：内蔵ヘルプドキュメントとナレッジベースに基づく RAG 質問応答。キーワードとシナリオに応じた操作提案を提供  
2. **第 2 段階**：データ処理と分析ワークフローをさらに支援し、反復作業を簡素化
### クロスプラットフォームとその他
