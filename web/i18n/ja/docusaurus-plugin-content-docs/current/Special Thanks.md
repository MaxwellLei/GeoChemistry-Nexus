# 🙏 特別な感謝

GeoChemistry Nexus はコミュニティのサポートによって存在しています。

## コントリビューター

GitHub Issue、ディスカッション、メールを通じて本プロジェクトに参加してくださったすべての方に感謝します。以下を含みますがこれらに限定されません：

* バグの報告、機能提案
* コード、テスト、ドキュメントの貢献
* 判別図、温圧計などのテンプレートの共有または共同構築

* 多言語ローカライズへの参加など

公開許可をいただいたすべてのコントリビューターは、プロジェクトのコントリビューターリストに掲載されます。個人ホームページや関連リンクがあれば、お知らせいただければ一緒に掲載いたします。
