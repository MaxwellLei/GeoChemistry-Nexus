# 🔢 自定义函数

对于地质温度计的计算，我们也可以通过内置的自定义函数来直接计算。如下是我们目前内置的地质温度计计算函数：

| 函数                                                         | 参数数量 | 说明                                               |
| ------------------------------------------------------------ | -------- | -------------------------------------------------- |
| `Zircon_Loucks_et_al_2020(ti,p,aTiO2,aSiO2)`                 | 4        | 锆石 Ti 温度计算，主量, Loucks et al. (2020)       |
| `Zircon_Watson_and_Harrison_1983(SiO2,Al2O3,fe2o3,feo,mgo,p2o5,cao,k2o,na2o)` | 10       | 锆石 Zr 温度计算，主量, Watson and Harrison (1983) |
| 待编写...                                                    |          |                                                    |

> 需要注意的是：由于地质温度计中间计算过程的考虑以及温度计计算方法的扩展，函数将会随着版本的更新有一定的变动。使用前建议优先参考最新文档。
