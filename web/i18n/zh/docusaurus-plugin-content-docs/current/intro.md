# 🌍 简介

**GeoChemistry Nexus 是面向地球化学与岩石学研究者的一体化工作台**：把数据预处理、判别图解绘制、地温计与 CIPW 计算、PNG/SVG 导出收拢到同一界面；并通过云端模板生态，让图解与算法持续更新、便于分享。

让我们在 **5 分钟** 内开始使用 **GeoChemistry Nexus** ⏱️。

## 前置条件 🛠️

需要提前说明的是：**软件仅适用于 Windows 用户**。

目前，尚无针对 Linux 和 Mac 的详细说明。建议优先使用虚拟机运行程序。我们计划在未来迁移到 `Avalonia` 以实现完全的跨平台兼容性。 📅

:::tip

**Windows 用户**：请确保你的操作系统版本至少为 Windows 7。

:::

## 以此开始 🚀

[点击这里](https://github.com/MaxwellLei/GeoChemistry-Nexus/releases) 访问下载页面。选择合适的版本，下载安装。然后，双击运行 `GeoChemistry Nexus.exe` 🖱️即可。
