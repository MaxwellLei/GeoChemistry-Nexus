# 🌍 简介

让我们在 **5 分钟** 内开始使用 **GeoChemistry Nexus** ⏱️。

## 前置条件 🛠️

在开始之前，请按照以下步骤操作：

1. **选择下载选项** 🌐：
   你可以通过以下方式之一获取所需内容：
   - **手动安装**：直接访问 [.NET 6 官方下载页面（点击这里）](https://dotnet.microsoft.com/en-us/download/dotnet/6.0) 并选择适合你操作系统版本的安装程序。
   - **跟随提示**：在软件执行过程中，跟随系统提示自动跳转到下载页面完成框架安装。
   - **无需手动配置：选择我们提供的独立包（无需额外安装运行时框架）**。
2. **确定操作系统类型** 💻：
   - 对于 Windows 系统，请首先确定你的操作系统是 `32位` 还是 `64位`，并选择相应的版本进行安装。

:::tip

**Windows 用户**：请确保你的操作系统版本至少为 Windows 7。

:::

## 以此开始 🚀

[点击这里](https://github.com/MaxwellLei/GeoChemistry-Nexus/releases) 访问下载页面。选择合适的版本，下载并解压。然后，双击运行 `GeoChemistry Nexus.exe` 🖱️。

### 软件包命名规范 📦：

**软件包命名遵循此格式：`软件名_缩写_版本_依赖_操作系统_架构`**

- 如果名称包含 `d`，表示该软件依赖 **.NET 6 SDK**，需要单独安装。 🔍
- 如果缺少 `d`，则软件已包含必要组件，无需进一步安装。 ✅

### Linux 和 Mac 用户注意事项 🐧🍏：

目前，尚无针对 Linux 和 Mac 的详细说明。建议优先使用虚拟机运行程序。我们计划在未来迁移到 `Avalonia` 以实现完全的跨平台兼容性。 📅
