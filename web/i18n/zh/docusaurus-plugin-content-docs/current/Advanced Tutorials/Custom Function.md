# 🔢 自定义函数

对于地质温度计计算，你还可以利用内置的自定义函数进行直接计算。目前内置的地质温度计计算函数如下：

| 函数 | 参数数量 | 描述 |
| :--- | :--- | :--- |
| `Zircon_Loucks_et_al_2020(ti,p,aTiO2,aSiO2)` | 4 | 锆石 Ti 温度计算，主量元素，Loucks et al. (2020) |
| `Zircon_Watson_and_Harrison_1983(SiO2,Al2O3,fe2o3,feo,mgo,p2o5,cao,k2o,na2o)` | 10 | 锆石 Zr 温度计算，主量元素，Watson and Harrison (1983) |
| 待定... | | |

> **注意：** 出于对中间计算过程和温度计计算方法扩展的考虑，这些函数可能会随着版本更新而更改。建议在使用前查阅最新文档。
