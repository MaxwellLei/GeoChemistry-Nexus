# 🛣️ 路线图

# 当前

- [x] 持续更新常用的地质和地球化学图表
- [ ] 持续更新常用的地质温度计和地质压力计
- [ ] 地质年代学数据绘图

# 计划中

- [ ] **研究社区建设**：我们将建立一个专门的研究社区，支持研究人员上传和分享自制的科学图表。社区还将配备问题反馈区和论坛讨论模块，以促进成员互动、经验分享和解决方案改进。
- [ ] **扩展地球化学计算工具**：根据用户需求，我们将不断更新和完善与地球化学相关的科学计算方法，提供更丰富的计算功能，以支持多样化的科研应用场景。
- [ ] **机器学习数据分析**：未来的软件版本将引入常用的机器学习算法来辅助地球化学数据分析，为研究人员提供高效智能的数据处理能力。
- [ ] **“新图表”风格机器学习模型支持**：随着大数据和机器学习的快速发展，许多模型已成为新的“判别图”工具。软件将内置可扩展接口，支持快速加载和训练此类模型。研究人员可以直接应用“新图表”进行分析和判别，无需掌握机器学习或 Python 等技术，避免了复杂的环境配置。
- [ ] **AI 智能助手**：利用内置的帮助文档和知识嵌入大模型技术，AI 助手将在第一阶段能够根据研究人员提供的关键词和需求生成针对性的解决方案；在第二阶段，它将进一步实现自动化数据处理和分析，简化科研工作流程并提高效率。
