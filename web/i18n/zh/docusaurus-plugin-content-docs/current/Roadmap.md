# 🛣️ 路线图

GeoChemistry Nexus 的目标不只是一个「画图软件」，而是逐步建成 **数据 · 图解 · 计算 · 协作** 一体化的地球化学与岩石学科研工具链。

## ✅ 已完成


### 核心工作流
- [x] **主页**：快捷链接、实用小组件
- [x] **数据预处理**：主量氧化物列识别，Fe 价态估算 / 反算，异常值、缺失值、检出限处理策略
- [x] **判别图解**：基于模板库绘制三角图、散点图、蛛网图等；支持数据投影与样式编辑
- [x] **地温计（GTM）**：内置多矿物计算模板及类 Excel 自定义函数
- [x] **CIPW 标准矿物计算**：全岩主量数据 norm 计算与结果导出
### 平台能力
- [x] **模板生态**：官方模板云端动态更新；JSON / ZIP 格式自定义模板创建、导入与打包分享
- [x] **多语言**：界面本地化（中 / 英 / 德等）；图解模板多语言一键切换
- [x] **导出与集成**：PNG / JPG / BMP / SVG 导出；支持与 CorelDRAW、Inkscape、Adobe Illustrator 等第三方软件协作

## 🔭 中长期愿景
### 机器学习辅助分析
引入常用机器学习流程，辅助地球化学数据判别与分类，输出模型评估、变量重要性及 ROC 曲线、混淆矩阵等可视化结果——**无需用户自行配置 Python 环境**。
### 「新图解」式判别模型
支持加载可扩展的机器学习判别模型（如基于大数据训练的新型判别图），让研究者直接应用「新图解」进行分析，而不必掌握完整的 ML 开发栈。
### AI 科研助手（分阶段）
1. **第一阶段**：基于内置帮助文档与知识库的 RAG 问答，根据关键词与场景给出操作建议  
2. **第二阶段**：进一步辅助数据处理与分析流程，简化重复性操作
### 跨平台与其他
- [ ] 评估 Avalonia 等方案，逐步实现 Linux / macOS 支持（当前以 Windows 为主）
- [ ] 持续优化性能、模板格式与开发者扩展接口
