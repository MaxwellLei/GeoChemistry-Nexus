## 🌟 计算方法和功能需求

在地球化学领域，计算方法的准确性和效率至关重要。**如果你在地球化学计算方面有深入的知识和经验，或者你的研究涉及特定的计算方法，我们诚挚邀请你为我们的项目做出贡献**。你的专业知识将有助于增强我们软件的计算能力，确保它更好地满足研究社区的需求！你的参与也将为新用户提供宝贵的支持，帮助他们充分利用我们的工具。

## 💻 开发者

我们的软件使用 `C#` 开发，基于现代 `MVVM` 架构和 `WPF` 技术，确保了出色的可维护性和可扩展性。此外，我们计划在未来引入 `Python` 来实现某些功能，以实现更灵活多样的功能。**如果你是一位经验丰富的 `C#` 或 `Python` 开发者，我们热烈欢迎你加入我们的项目团队。无论你是想贡献代码、设计用户界面还是进行功能测试，你的参与都将使我们的软件变得更好**。
