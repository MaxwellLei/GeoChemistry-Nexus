# 🌡️ 地质温度计

软件中的地质温度计页面简写为 **GTM**。此页面目前包括锆石、闪锌矿、石英、磷钇矿、绿泥石和黑云母等矿物的地质温度计计算。

:::tip

请注意，当前的地质温度计模块采用临时 UI。随着温度计功能的增强，它将被调整。

:::

## 🌟 特性

我们的地质温度计模块现在采用类似 Excel 的界面，显着降低了学习曲线。可以通过两种方式进行计算：

1.  **直接在单元格中使用函数**，与使用 Excel 函数相同。我们为各种温度计预定义了几个计算函数。有关具体函数详情，请参阅高级教程中的“自定义函数”部分。
2.  **使用内置地质温度计模板**，这种方式更直接、更快速，无需编程知识。

![gtm_ui](imgs/gtm_ui.png)

## 使用示例

### 自定义函数

自定义函数的使用与 Excel 一致；只需在单元格中输入类似 `=FuncName(args)` 的格式即可调用它们。

具体自定义函数参考请参阅高级教程。

### 使用内置模板

首先，在地质温度计模块内，从菜单栏中选择相应的计算模板：

![gtm_select_m](imgs/gtm_select_m.png)

随后会出现一个弹窗，询问是否为计算创建一个新工作表。

如果你当前的工作表包含不想覆盖的数据，请选择 **确定**。反之，如果你想覆盖当前工作表，请选择 **取消**。

![Notification](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/GTM_New_Sheet.png)

接下来，会出现一个温度计的示例模板。用户只需填写相应的数据，然后使用“点击并拖动”填充柄（类似于 Excel）自动完成计算（如下面的动画示例所示）。

![](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/Example.gif)
