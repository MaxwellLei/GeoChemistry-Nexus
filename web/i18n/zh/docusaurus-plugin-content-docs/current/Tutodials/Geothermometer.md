# 地质温压计

在早期的版本中该模块简称为 **GTM**。后来改成地质温度计，到如今的正式定名为：地质温压计。

早期版本包含锆石、闪锌矿、石英、磷钇矿、绿泥石和黑云母等矿物的地质温度计计算。新版本的地质温压计还在更新适配中，不久将会同步。

## 🌟 特性

地质温压计模块设计之初就**仿照 Excel 的形式**，来降低用户的上手成本。**主要是分为两种形式来使用温压计**：

1. **内置云模板**：默认情况下**内置云模板**用户不可修改，由官方维护更新。可以在不更新软件的情况下，联网获取更多最新的地质温压计模板。研究人员可以搜索并选择适合自己的模板，并按照指南内容进行数据填写和计算。**这种方式更直接、更快速，无需编程知识**。
2. **独立窗口**：研究人员可以通过在菜单栏，文件 -> 打开独立表格，来打开独立窗体。在独立窗体中，研究人员被允许自定义编程使用不同的地质温压计进行组合计算。具体不同温压计对应函数，请参见进阶教程部分。**这种方式更复杂，相对灵活**。

当然，地质温压计模块也允许研究人员创建自己的自定义温压计。通过打包成地质温压计包可以导出分发给其他研究人员使用。

## 地质温压计使用示例

### 使用内置模板

首先，在地质温度计模块内，从菜单栏中选择相应的计算模板：

![gtm_ui](/img/v0.7.1/geothermobarometer2.webp)

随后会出现一个二次确认侧边栏弹窗，确定后之前的模板计算会被重置覆盖（如果在之前有选择别的温压计计算的话）。

![gtm_ui](/img/v0.7.1/geothermobarometer3.webp)

接下来，会出现一个温度计的示例模板。用户只需填写相应的数据，然后使用“点击并拖动”填充柄（类似于 Excel）自动完成计算（如下面的动画示例所示）。

![](https://geo-1303234197.cos.ap-hongkong.myqcloud.com/V0_5_0_0/Example.gif)

当然，新版本温压计已支持查看内部计算详情。研究人员可以通过鼠标选中对应数据行，在底部查看具体计算中间结果详情。

![gtm_ui](/img/v0.7.1/geothermobarometer4.webp)

### 独立窗口-自定义函数

如前面所讲述，可以通过在菜单栏，文件 -> 打开独立表格，来打开独立窗体。

独立窗口表格是一个空表格，包含有一些简单的工具条按钮。

![gtm_ui](/img/v0.7.1/geothermobarometer5.webp)

在使用独立窗口表格就和我们日常**使用 Excel 表格一样，可以输入任意数值，通过输入`=函数名称(传递参数)` 来调用函数**。

不同的是，我们的地质温压计计算被注册为了对应的计算函数，研究人员可以根据需求更灵活的自定义内容计算。包括但是不限于多温压计结果对比。在默认的模板情况下，无法实现同时计算多个温压计结果（除非你一个模板一个模板计算）。但是在独立窗口表格下，你可以直接在同一个表格里面定义并使用多个地质温压计函数来获取计算结果。

:::note

**在独立窗口表格下，无法查看对应数据行的中间计算过程**。

:::
