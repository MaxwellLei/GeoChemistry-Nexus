# 🏠 首页

# 目标

首页的最初设计目标是整合地球科学领域常用网站的快速访问，并提供实用的小部件以支持相关操作和计算，最终提高研究效率。

![plot_home_ui](/img/v0.6.1/tutorial_home1.png)

首页主要显示 **快速链接** 和 **小部件**。你可以右键点击打开菜单栏查看相关操作。

关于快速链接，我们的设计理念是用户可以添加地质学和地球化学中常用的专业网站。此外，软件还包含内置推荐站点。如果你希望你的网站被添加到首页推荐中，请通过“关于”页面联系我们。

目前，首页默认显示系统日期和时间。它还具有一个搜索栏，可以搜索和匹配用户添加的快速链接和小部件，以便快速查找和导航。

小部件的设计目标是提供简单、独立且基础的计算或操作功能，以补充常用的外围工具，方便用户日常使用。

:::info

随着软件的不断更新，首页功能将变得更加实用。

我们欢迎你提供宝贵的反馈，帮助我们提高软件的可用性和便利性。 🌹

:::
