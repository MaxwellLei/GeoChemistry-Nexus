# 主页

# 目标

首页是负责整合地球科学领域常用网站的快捷访问入口，并提供实用的小工具以支持相关操作与计算，从而提升科研效率。

我们的设计目标是尽可能多地整合地质、地球化学及地球科学领域的网站与工具，通过首页快捷访问链接起来，构建一个互联互通的便捷通道。

![plot_home_ui](/img/v0.7.1/intro_user_interface_example1.webp)

首页主要功能是：**公告**，**快速链接** 和 **小部件**。

* **公告**：负责接受服务器通知
* **快速链接**：负责整合地球科学领域入口，分为两个大类。**推荐链接**和**个人链接**，推荐链接个人无法修改，是通过服务器进行内置推荐。当然，如果你希望你的网站被添加到首页推荐中，请通过“关于”页面联系我们。个人链接是用户个人进行添加删除管理个人的喜好链接。
* **小部件**：小部件的设计目标是提供简单、独立且基础的计算或操作功能，以补充常用的外围工具，方便用户日常使用。以后会做成可拓展的插件形式，目前无法自由拓展。

:::info

随着软件的不断更新，首页功能将变得更加实用。

我们欢迎你提供宝贵的反馈，帮助我们提高软件的可用性和便利性。 🌹

:::
