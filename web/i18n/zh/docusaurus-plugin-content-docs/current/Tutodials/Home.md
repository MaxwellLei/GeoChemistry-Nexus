# 🏠 主页

# 目标

主页的初始设计目标是整合地球科学领域常用网站的快捷入口，并提供实用的小组件，以支持相关操作与计算，最终促进科研效率的提升。

![plot_home_ui](imgs/plot_home_ui.png)

主页主要是承载：**快捷链接**和**小组件**。

当然，目前情况下，默认软件显示当前系统日期，时间。提供一个搜索栏，该搜索栏目前只是一个简单的链接跳转。在该搜索栏搜索的内容会跳转本地浏览器的谷歌搜索进行搜索。

> 搜索功能暂不可用

:::info

随着后续软件的更新，主页的功能将会更加实用。

我们欢迎大家在使用过程中提供宝贵的反馈，以提高软件的可用性和便利性。🌹

:::
