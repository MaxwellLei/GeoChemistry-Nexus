# ⚙️ 设置

软件的设置部分非常多样化，并将随着功能的扩展不断更新更多的个性化选项。

## 常规程序设置

设置主要分为四个部分：文件路径、国际化与个性化、消息通知和其他。下面是对主要设置的简要说明：

![tutorial_settings1](/img/v0.6.1/tutorial_settings1.png)

### 🌍 语言 & 外观

-   **语言**：目前支持简体中文、英语、德语、西班牙语、日语、韩语和俄语。
-   **启动背景图像**：软件启动时显示的背景图像。

:::info

关于语言支持，如果你想帮助进行你所在国家的本地化，我们热烈邀请你加入我们的翻译工作！目前的翻译大多由 AI 辅助，这意味着某些部分可能不准确——非常需要并感谢你的帮助。

:::

### 🗨️ 消息通知

-   **自动关闭时长**：软件运行期间消息通知弹窗自动关闭的时长。请注意，错误和警告消息需要手动关闭。

### ⚙️ 其他

-   **随系统启动**：软件将在系统启动时自动启动。
-   **开发者模式**：此模式为开发者提供调试功能。建议非开发者不要启用，以免误操作。
-   **自动检查更新**：软件将在启动时自动检查更新；但是，由于服务器问题，此功能暂时禁用，将在未来版本中恢复。
-   **退出方式**：软件可以通过两种方式退出：直接退出或最小化到系统托盘。如果你选择最小化到托盘，你可以右键点击托盘图标并选择退出来关闭软件。

## 绘图模块设置

绘图模块设置专门针对绘图模块内的功能，包括但不限于：

*   第三方程序路径配置
*   默认第三方程序
*   图表默认加载语言
*   图表模板和分类结构的自动更新
